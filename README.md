# Student Result Management System (Assembly Language)

## Description
This is a menu-driven Student Result Management System written in x86 Assembly Language.

## Features
- Add student ID
- Validate digit-only input
- Store marks
- Calculate total and average
- Display results

## Tools Used
- Language: x86 Assembly
- Assembler: MASM / TASM / emu8086
- Platform: DOS

## How to Run
1. Open the code in emu8086
2. Compile and run `main.asm`
